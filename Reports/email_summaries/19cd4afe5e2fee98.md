CUDA 13.2 Release Notification
The NVIDIA CUDA 13.2 release is now available, featuring important platform updates, language features, and tooling support.  
Please review the details below to ensure your development environment and workflows remain up to date. 
Added New Features in Release 13.2 
 	- CUDA Tile support for compute capability 8.X (Ampere and Ada) GPU architectures. 	- Unified CUDA Toolkit for Arm targets. 	- Support for recursive functions and closures in cuTile Python. 	- Support for emulated FP64 in math libraries 	- Numba-CUDA debugging to developer tools. 	- CUDA Graphs to cuda.core.
