# Настройки подключения по SSH (Удаленный доступ)

В этом файле собраны все данные для удаленного управления вашими устройствами. Так как этот файл находится в папке `Sync`, он всегда будет под рукой и на телефоне, и на ПК, и на Chromebook.

---

## 1. Как подключиться ко МНЕ (К главному ПК на Windows)
Если вы находитесь в Termux на телефоне или в терминале Chromebook и хотите управлять главным компьютером (где запущены основные агенты):

* **Пользователь:** `anton`
* **Локальный IP (если вы дома в одном Wi-Fi):** `192.168.1.12`
* **VPN IP (Tailscale - доступ из любой точки мира):** `100.72.214.118`

**Команда для подключения:**
```bash
# Если вы дома:
ssh anton@192.168.1.12

# Если вы далеко от дома (через Tailscale):
ssh anton@100.72.214.118
```
*(При первом подключении напишите `yes`, затем введите пароль от вашей учетной записи Windows).*

---

## 2. Как подключиться к ТЕЛЕФОНУ (Termux)
Если вы сидите за ПК (Windows) или Chromebook и хотите поковыряться в Termux:

* **Пользователь:** не требуется указывать (или любой)
* **Порт:** `8022`
* **IP телефона:** `192.168.1.2` (IP может меняться, проверьте командой `ifconfig` в Termux)

**Предварительно в Termux нужно один раз запустить сервер и задать пароль:**
```bash
passwd  # задать пароль
sshd    # запустить сервер
```

**Команда для подключения с ПК/Хромбука:**
```bash
ssh -p 8022 192.168.1.2
```

---

## 3. Как подключиться к CHROMEBOOK (Penguin)
Если вы хотите зайти в Linux-контейнер Хромбука с ПК или телефона:

* **Пользователь:** ваше имя пользователя в Linux (обычно совпадает с почтой до @)
* **IP Хромбука:** `192.168.1.14` (IP может меняться)

**Команда для подключения:**
```bash
ssh ИМЯ_ПОЛЬЗОВАТЕЛЯ@192.168.1.14
```
*(В настройках ChromeOS Linux должен быть включен проброс портов или разрешен SSH-доступ).*
