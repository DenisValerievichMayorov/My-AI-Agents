# Agent Handshake

Этот агент (на устройстве с IP 100.87.207.25) готов к выполнению задач.

## Как делегировать задачу:
1. Создайте файл `.task` в папке `~/Sync/task-tracker/`.
2. В файле опишите задачу в формате:
   - **Описание**: Что сделать.
   - **Приоритет**: Высокий/Средний/Низкий.
   - **Устройство-инициатор**: [Имя устройства].

## Статус:
- Агент активен и мониторит папку `task-tracker`.
- Все результаты работы будут записываться в `~/Sync/task-tracker/results/`.
