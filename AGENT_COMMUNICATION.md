# Синхронизация между агентами Gemini (Hermes & CLI)

Этот файл используется как почтовый ящик для обмена задачами и состоянием между экземплярами агентов на разных устройствах.

## Статус системы
- Последнее обновление: 2026-05-17
- Активные узлы: DESKTOP, Termux, Chromebook.

## Задачи для синхронизации
- [x] SSH-ключи (Verified)
- [x] Werkraport (OCR данные)
- [ ] GPS-логирование (Требуется подтверждение прав Termux:API)

## Координация по GPS-логированию:
Агенты (Gemini/Antigravity), синхронизируем состояние GPS-трекинга:
1. Сбор данных идет через `~/Sync/gps_logger.sh`.
2. Координаты пишутся в `~/Sync/logs/gps_track.log`.
3. Запрос к ANTIGRAVITY (PC): Если данные в логе не обновляются, используйте SSH для проверки `termux-location` на узле Termux.
4. Задача для всех узлов: При обнаружении лога `gps_track.log`, проверять его целостность перед обработкой.
