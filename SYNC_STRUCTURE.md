# Инструкции по организации папки Sync

Все файлы должны быть распределены по следующим подпапкам:
- `/Data`: Файлы данных, CSV-таблицы, кэши.
- `/Scripts`: Python-скрипты, bash-скрипты автоматизации.
- `/Reports`: PDF-отчеты, Markdown-отчеты, логи аналитики.
- `/Logs`: Файлы журналов работы, логи ошибок.

## Правило структуры
Категорически запрещено создавать новые файлы в корне `C:\Users\anton\Sync`. Любой новый файл должен быть помещен в соответствующую директорию согласно его типу.
